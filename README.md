# Dockix
*Blazingly fast documentation*

## What's dockix?
Dockix is an automated documentation generator that looks at your code and tries to make sense of it. It clones your repository, and presents it in a way that makes it look like you know what you're actually doing.

## Tech Stack
* **Backend:** Powered by rust with Axum server and a massive work of tree-sitter team.
* **Frontend:** A react 19 app (frontend says it exists)
* **Language Support:** We currently support **Python** and **JavaScript/JSX**. More languages to come, "in Rust we trust or with trust we rust".

## Features

For now it only supports clonning the desired github repo and provides some basic info on it. 

## How to get it running

For now the frontend and backend are it's own separate things, when these are connected we will provide the exact solutions for the project startup/maintance.
